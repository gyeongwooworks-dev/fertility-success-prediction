# 난임 환자 임신 성공 여부 예측 AI 모델 개발 해커톤

>  | LightGBM + CatBoost Ensemble | **AUC 0.74197**

---

## 프로젝트 개요

2023년 합계출산율 0.72명(OECD 최하위), 국내 연간 난임 진단 25만 건 이상이라는 사회적 배경 속에서,
256만 건의 난임 시술 기록을 바탕으로 임신 성공 여부를 예측하는 AI 모델을 개발했습니다.

단순 전처리 기술을 넘어, 임상적 도메인 지식(PGD/PGS 결측 해석, 난자 출처별 성공률 차이 등)을 피처 설계에 반영하여
**"수혜자의 나이가 아닌 난자의 나이가 임신 성공의 핵심 변수"** 라는 인사이트를 모델에 구현했습니다.

## 데이터 설명

| 항목 | 내용 |
|------|------|
| 출처 | 데이스쿨 데이콘 해커톤 제공 데이터 |
| 규모 | 약 256만 건의 난임 시술 기록 |
| 타겟 | 임신 성공 여부 (이진 분류) |
| 전체 임신 성공률 | 약 25~30% (클래스 불균형) |

**주요 변수 카테고리**

| 카테고리 | 변수 예시 |
|----------|-----------|
| 시술 정보 | 시술 유형 (IVF/ICSI/IUI/배반포이식), 시술 주기 횟수 |
| 배아 정보 | 이식된 배아 수, 배아 이식 경과일, 냉동 배아 사용 여부 |
| 난자 정보 | 난자 출처 (자가/기증), 수집된 신선 난자 수 |
| 유전자 검사 | PGS 시술 여부, PGD 시술 여부 |
| 환자 정보 | 나이, 총 임신 횟수, IVF 시술 횟수 |

## 데이터 전처리

### 결측치 처리

| 변수 유형 | 처리 방법 |
|-----------|-----------|
| 유전자 검사 컬럼 (PGS, PGD 등) | 0으로 대체 (미시술 의미) |
| 나이 수치 | Train 중앙값(37세)으로 대체 |
| 배아/난자 카운트 | 0 대체 + 결측 여부 binary flag 추가 |
| 날짜 기반 컬럼 | 시술 수행 여부 binary flag로 변환 |

### 이상치 처리

별도 IQR 클리핑·로그 변환 없이 트리 모델의 비선형 분기가 이상치를 자동으로 처리하도록 위임. 다만 비율 파생변수(배아_잉여율, 수정률)는 정의상 [0, 1] 범위로 클리핑.

> **데이터 누수 방지**: 결측값 대체 기준(수치형 → 0, 범주형 → '알 수 없음')은 상수로 고정. fold 간 통계 기반 보정(평균·중앙값 imputation) 미사용.

### 인코딩

- 범주형 변수: LightGBM · CatBoost 네이티브 category dtype 처리 (별도 OHE 없음)
- 멀티라벨 변수 (`배아 생성 주요 이유`): sklearn `MultiLabelBinarizer`로 5개 이진 컬럼 분해 (클래스는 Train 기준으로 고정)
- 결측 범주: `알 수 없음`으로 통일

## 피처 엔지니어링

**도메인 파생 변수 (`src/preprocessing.py`)**

| 파생 변수 | 설명 |
|-----------|------|
| `Age_Median` | 나이 구간 문자열 → 구간 중앙값 수치화 |
| `Age_Risk_Group` | WHO·ACOG 기준 4단계 임상 위험군 (Normal / High_Early / High_Extreme / High_Reversal) |
| `유전검사_시행여부` | PGS·PGD 결측 = 미시행으로 해석, 어느 하나라도 기록 시 1 |
| `기증난자_여부` | 난자 출처 == '기증 제공' binary flag |
| `기증난자×나이` | 기증 난자 사용 시에만 나이 값 활성화 (고령 환자 성공률 회복 포착) |
| `자가난자×나이` | 자가 난자 사용 시에만 나이 값 활성화 (수혜자 나이 페널티 격리) |
| `배아_잉여율` | (총 생성 배아 수 − 이식 수) / 총 생성 수 — 배아 품질 간접 지표 |
| `Age×배아수` | 나이 × 총 배아 수 교호작용 — 고령에서 배아 다수일 때 효과 포착 |
| `수정률` | 혼합된 난자 수 / 수집된 신선 난자 수 — 정자 품질 간접 지표 |
| `저장배아_보유여부` | 저장된 배아 수 > 0 — 다음 시도 자원 보유 여부 |
| `총_시술횟수` | IVF 시술 횟수 + DI 시술 횟수 통합 |
| `기증정자_사용여부` | 기증자 정자 혼합 기록 유무 — 결측 = 미사용으로 해석 |

**팀원 설계 임상 피처 (`src/features.py`)**

| 파생 변수 | 임상 근거 |
|-----------|-----------|
| `배반포_5일차_이식` | 수정 후 5일 배반포 이식은 성공률 우위 (Roque 2013) |
| `젊은_고효율_이식` | Age ≤ 37 + 배아 잉여율 > 0.3 + 이식 ≥ 1 — 임상 최적 조건 |
| `이식경과일_구간` | no_transfer / early(≤3일) / blastocyst(5일) / other 구간화 |
| `동결_기증_복합` | 동결 배아 + 기증 난자 동시 사용 — 고령 성공률 회복의 핵심 조합 |
| `고령_반복시술` | Age ≥ 40 + 총 시술 횟수 ≥ 2 — 임상 난이도 최고 환자군 |

**핵심 인사이트:** 만 43-44세에서 자가 난자 성공률 ~0% vs 기증 난자 55.6% → 난자 출처와 나이의 교호작용이 결정적

## 모델링

### 모델 선택 이유

| 이유 | 선택 방향 |
|------|-----------|
| 다중공선성이 매우 큰 데이터셋 | 트리 모델 (상관 변수 자동 처리) |
| 범주형 변수 다수 (나이·시술유형·난자출처 등) | CatBoost (내장 범주형 처리) |
| 256만 건 대용량 + 고차원 피처 | LightGBM (속도 우위) |
| 비선형 교호작용 풍부 | 트리 분기로 자동 포착 |

### 사용 모델

- LightGBM
- CatBoost
- 하이퍼파라미터 튜닝: Optuna
- 클래스 불균형 처리: `scale_pos_weight` (음성:양성 비율 자동 계산, SMOTE 등 resampling 미사용)

## 실험 결과

> 76개 피처, 25-Fold 교차검증 기준

| 모델 | AUC |
|------|-----|
| LightGBM | 0.7404 |
| CatBoost | 0.7407 |
| **LightGBM + CatBoost Ensemble (50:50)** | **0.7409** |

IVF 임상 데이터 기반 ML 모델의 일반적 성능 범위는 AUC 0.65~0.80 (McLernon 2016, Hum Reprod 31:572) — 본 모델은 학계 기준 우수 수준에 해당합니다.

## 최종 모델

**LightGBM + CatBoost Ensemble (50:50)**

| 지표 | 값 |
|------|----|
| OOF AUC | **0.7409** |
| Recall | 0.760 |
| Precision | 0.392 |
| F1 Score | 0.518 |

SHAP 분석을 통해 모델 예측의 임상적 근거를 제시했습니다.

**도메인 기반 핵심 피처 11종**

| # | 변수명 | 임상 근거 |
|---|--------|-----------|
| 1 | Age_Median | 나이 → 구간 중앙값 수치화 |
| 2 | Age_Risk_Group | WHO/ACOG 4단계 임상 위험군 |
| 3 | 유전자검사시행여부 | PGS·PGD 결측 = 미시행 통합 |
| 4 | 기증난자×나이 | 45~50세 성공률 격차 17.6%p |
| 5 | 자가난자×나이 | 수혜자 나이 영향 격리 |
| 6 | 배아잉여율 | 배아 품질 간접 지표 |
| 7 | Age×배아수 | 고령+다수배아 교호작용 |
| 8 | 수정률 | 정자 품질 간접 지표 |
| 9 | 저장배아보유여부 | 다음 주기 자원 보유 |
| 10 | 총시술횟수 | IVF+DI 시술 이력 통합 |
| 11 | 기증정자사용여부 | 결측=미사용 임상 재해석 |

**한계점**

- 체질량·생활습관 등 신체 계측 지표 미반영
- 기저 질환(당뇨·고혈압 등) 데이터 부재
- 면역·미세환경 변수 없음 (측정 불가 요인 30~40%)
- False Positive 과다: Recall 76%, Precision 39% → 환자 기대-실망 사이클 위험
